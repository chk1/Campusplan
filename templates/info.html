WWU Campusplan

Web App Version 2.0

Die WWU Campusplan Webapp wird als Open Source Projekt vom Institut für Geoinformatik und der Pressestelle gepflegt. Der Code befindet sich auf GitHub – Verbesserungsvorschläge von programmierfreudigen per Pull Request werden gern angenommen.
